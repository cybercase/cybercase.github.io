I’m happy to announce my first macOS Application:
Mutant: a GraphQL Explorer App for macOS

It’s a handy tool for GraphQL developers used to jump between GraphiQL and GraphQL-Voyager. Mutant combines the features of these two great tools in a lightweight macOS app.

There’s still a lot of work to do, so I’m running a beta test to gather some feedback, and if you are a GraphQL developer I can’t wait to hear from you.

You can download here the latest version of the app and drop your thoughts to stefano@brilli.me.
